                      ⚡ OPCIÓN 1: RÁPIDO (30 MIN)

════════════════════════════════════════════════════════════════════════════════

  1️⃣  Abre terminal PowerShell COMO ADMINISTRADOR
      • Windows + X → PowerShell (Admin)
      • O busca "PowerShell" → click derecho → "Run as Administrator"

  2️⃣  Ejecuta comando:

      cd c:\Proyectos\emma\security-research
      powershell -ExecutionPolicy Bypass -File INSTALAR_HERRAMIENTAS.ps1

  3️⃣  El script pregunta (responde "s" o "n"):
      • Instalar Burp Suite? → s (prioritario)
      • Instalar jq? → s (recomendado)
      • Instalar OWASP ZAP? → n (opcional)
      • Instalar Postman? → n (opcional)

  4️⃣  Espera 10-15 minutos
      • Descarga Burp Suite (~150 MB)
      • Instala todo automáticamente

  5️⃣  ¡LISTO! Todas las herramientas instaladas ✅
