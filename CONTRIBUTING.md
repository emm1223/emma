# Contribuir a EMMA

Gracias por querer contribuir. Sugerencias rápidas:

1. Haz un fork del repo.
2. Crea una rama descriptiva: `feature/tu-cambio` o `fix/descripcion`.
3. Añade pruebas cuando aplique y documentación mínima.
4. Envía un PR con descripción clara y pasos para reproducir.

Formato de commit recomendado: `type(scope): breve descripción` (ej.: `feat(readme): mejorar hero`)
