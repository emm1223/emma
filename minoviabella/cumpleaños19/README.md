# 🎂 Carta de Cumpleaños Digital - 19 Años 🎂

Una página web interactiva y hermosa para celebrar el cumpleaños de tu amor.

## 📁 Estructura

```
cumpleaños19/
├── index.html          # Página principal
├── styles.css          # Estilos y animaciones
├── script.js           # Funcionalidad interactiva
├── fotos/              # Carpeta para las fotos
│   ├── foto1.jpg       (Agrega aquí las fotos)
│   ├── foto2.jpg
│   ├── foto3.jpg
│   ├── foto4.jpg
│   ├── foto5.jpg
│   └── foto6.jpg
└── README.md           # Este archivo

```

## 🎯 Cómo usar

### 1. Agregar Fotos
- Coloca tus fotos en la carpeta `/fotos`
- Guarda las fotos como: `foto1.jpg`, `foto2.jpg`, `foto3.jpg`, etc.
- También puedes usar `.png` o `.webp`

### 2. Personalizar el Mensaje
- Abre `index.html` con un editor de texto
- Busca las secciones que dicen "Mi Amor, En Tu Día"
- Reemplaza el texto con tu mensaje personalizado

### 3. Cambiar la Fecha de Relación
- En `script.js`, busca la línea:
  ```javascript
  this.startDate = new Date('2025-10-17');
  ```
- Cambia a tu fecha (formato: YYYY-MM-DD)
- Actualmente está: 17 de octubre de 2025

### 4. Abrir la Página
- Haz doble clic en `index.html`
- O arrastra el archivo a tu navegador
- ¡Disfruta!

✅ **Botones interactivos** (Confeti, Besos, Música)

## 💡 Consejos

1. **Las fotos deben ser cuadradas** para verse mejor en la galería
2. **Usa imágenes de buena calidad** (no pixeladas)
3. **Puedes tener entre 3 y 6 fotos** máximo
4. **Abre en navegador moderno** (Chrome, Firefox, Safari)
5. **Comparte el link** si alojaste en un servidor web
