📸 AGREGAR TUS FOTOS AQUÍ

Nombre tus fotos exactamente así:
  • foto1.jpg
  • foto2.jpg
  • foto3.jpg
  • foto4.jpg
  • foto5.jpg
  • foto6.jpg

IMPORTANTE:
✓ Usa las extensiones: .jpg, .png, o .webp
✓ Las fotos deben ser cuadradas para verse mejor
✓ Puedes tener entre 3 y 6 fotos
✓ Usa imágenes de buena calidad (no pixeladas)

EJEMPLO:
[Arrastra tus fotos aquí]
fotos/
  ├── foto1.jpg
  ├── foto2.jpg
  ├── foto3.jpg
  ├── foto4.jpg
  ├── foto5.jpg
  └── foto6.jpg

¡Listo! Las fotos aparecerán automáticamente en la galería.
